# Repo-Pack (Starter Kit)

Zweck:
Dieses Paket gehört ausschließlich ins Git-Repository (Repo-Root). Es erzwingt Repo-Hygiene und Qualitäts-Gates.

Anwendung (Kurz):
1) Repo anlegen/klonen
2) Inhalt dieses ZIPs in das Repo-Root entpacken
3) Commit + Push
4) Branch Protection aktivieren (Required status checks)

Was drin ist (typisch):
- GitHub Actions: CI, CodeQL, Dependency Review, Gitleaks, Release
- PR/Issue Templates, CODEOWNERS, .editorconfig, Dependabot

Was es NICHT ist:
- Kein Projekt-Knowledge-Hub
- Kein ChatGPT-Backoffice

Merksatz:
Repo-Pack kontrolliert, was ins Repo darf.

---

Hinweis: Dieses Repo-Pack kann für Traumtänzer (Modus Mono) verwendet werden (Repo-Hygiene/Gates).
